测速设置

测速逻辑：默认并发固定为 5（每个连接启动 1 个 xray），然后依次测速所有网站；已存在(连接+网站)结果会自动跳过不重复测。IP属地测试固定取当前网站速度最快的前 3 个代理，通过 api.myip.com 回填国家。
